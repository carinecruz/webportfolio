E-commerce API

This is an API hosted in Heroku for the e-commerce website project. This includes schemas for users, orders, products, payments and reviews. Each schema has CRUD (Create, Read, Update, and Delete) operations, inclucing search functions. The product schema includes database image upload.

The front end is built with vanilla javascript and bootstrap to demonstrate basic API functionalilites.

Technical Stack: MongoDB, Rest, Express.JS, Node.js
E-commerce Website

This is a fullstack project. The e-commerce API is integrated with the front end system built using React.js and CSS. Functionalities include:

  • User registration
  • Login
  • Update profile and change password
  • product browsing (admin/user/guest)
  • Add to cart, including cart modification
  • Cart check out
  • Payment
  • Image upload for products
  • Create, update, delete products
  • Displaying featured products and top 6 best-selling items

Technical Stack: MongoDB, Express.js, Rest, Node.js, React.js, CSS
